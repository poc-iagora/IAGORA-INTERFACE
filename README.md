
## Quick start

- `npm i`
- [Download from Github](https://github.com/creativetimofficial//archive/master.zip).
- [Download from Creative Tim](https://www.creative-tim.com/product/?ref=adsr-github-readme).
- Install with [Bower](https://bower.io/?ref=creativetim): `bower install `.
- Clone the repo: `git clone https://github.com/creativetimofficial/.git`.

## Documentation

The documentation for the Argon Design System React is hosted at our [website](https://demos.creative-tim.com//#/documentation/overview).
